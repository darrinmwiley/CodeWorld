Welcome to the codebase :)

I'm actually not going to abandon you as much as an escape room might. 
Instead, I'm leaving you some documentation. It should help a little, but
you'll still need to figure out a thing or two to solve the level.

Some very basic things to know about the editor:

1. Press TAB to open, press ESCAPE to close
2. One of the buttons on the left pops the file explorer panel in / out, 
   if at any point you find yourself needing more space
3. On a lot of the files, you'll see a lock icon to the left of the line 
   numbers. that means the line is read only, and you can't edit it. 
4. For files that you can change, there is pretty good hotkey support. For
   instance, if you manage to accidentally break the code, you can spam 
   CTRL+Z to undo your changes. 

I'd recommend checking out all of the files as well, you wouldn't expect to
beat an escape room without reading all of the clues right?
