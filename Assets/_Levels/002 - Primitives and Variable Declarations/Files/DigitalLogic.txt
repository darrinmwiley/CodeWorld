Digital logic is a way of using diagrams to describe different logical
expressions like (a AND b), (x OR y), NOT(c)

Basically, each operator (AND, OR, NOT, etc..) has a specific shape that
represents it. We draw lines into the shape to denote inputs, and a single
line out of each shape for the resulting output.
