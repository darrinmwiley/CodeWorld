# Directory Purpose
Two-pane printer-oriented UI variant used for code-and-output style workflows.

## What Lives Here
- `Printer 2Pane.prefab` plus matching UXML/USS.
- Supporting toolbar/resize icon textures.

## Key Files
- `Printer2Pane.uxml`: layout definition for printer panel arrangement.

## How It Connects
- Can be instantiated by level scenes that need paired panes for coding/output interactions.

## Runtime Role (LV1/LV2/Server)
- Unity presentation shell for printer-focused interactions.

## Dependencies
- Shared UI theme assets and window interaction systems.

## Notable Flows
- Scene UI initialization binds pane content and interactions.

## Maintenance Notes
- Keep this folder focused on printer-specific UI composition.
